# ⚡ Super SSH —— 让 SSH 变得简单

## 一句话介绍

**Super SSH** 是一款基于 Web 的 SSH 可视化终端工具。无需记忆复杂命令，通过图形界面即可轻松连接远程服务器、管理文件，同时还能学习背后的 SSH 命令。

## 核心亮点

### 🖥️ 浏览器里的 SSH 终端
打开浏览器就能用，不依赖任何客户端。基于 xterm.js 打造的全功能终端，支持 256 色显示、窗口自适应，体验不输本地终端。

### 📁 像管理本地文件一样管理远程文件
- **拖拽上传**：把文件拖到浏览器里，自动上传到服务器
- **在线浏览**：双击进入文件夹、一键返回上级，操作直觉流畅
- **右键菜单**：下载、删除、复制路径，右键即达
- **实时进度**：文件传输进度一目了然

### 📋 边操作边学命令
这是 Super SSH 最独特的功能。你在界面上的每一个操作，都会自动生成对应的 SSH 命令并显示在左侧面板：

- 进入目录 → `cd /path/to/dir`
- 上传文件 → `scp "file" user@host:/path/file`
- 下载文件 → `scp user@host:/path/file ./`
- 删除文件 → `rm -rf /path`
- 新建文件夹 → `mkdir -p /path`

**点击命令即可复制**，让你在图形界面操作中自然学会 SSH 命令。

### 💾 连接信息管理
常用的服务器连接一键保存，下次直接点击连接，不用重复输入地址、端口、用户名。

### 🎨 个性化界面
支持自定义背景图片和透明度调节，让你的终端与众不同。

## 技术架构

- **后端**：Node.js + Express + WebSocket + ssh2
- **前端**：原生 JavaScript + xterm.js
- **通信**：WebSocket 实现终端实时交互，SFTP 实现文件传输
- **零依赖部署**：一个命令启动，浏览器即可访问

## 适用场景

- **运维人员**：快速连接多台服务器，高效管理文件
- **开发者**：本地开发机与远程服务器之间无缝传输代码
- **SSH 初学者**：通过可视化操作学习命令，降低学习门槛
- **团队协作**：统一工具，减少环境配置成本

## 快速体验

```bash
git clone <repo-url>
cd super-ssh
npm install
npm start
```

浏览器打开 http://localhost:3000，30 秒内连接你的第一台服务器。
